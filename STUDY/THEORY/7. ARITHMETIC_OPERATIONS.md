## Toán tử trong COBOL

### ADD - Phép cộng

---

```cobol
ADD {CORRESPONDING/CORR} {Identifier1/literal1} TO {identifier2} GIVING {Identifier3} {ROUNDED}
[ON SIZE ERROR imperative-statement-1]
[NOT ON SIZE ERROR imperative-statement-2]
END-ADD
```

Trong đó:

- `CORRESPONDING / CORR ... TO ...`: Cộng các trường có cùng tên trong hai nhóm dữ liệu khác nhau.
  ```bash
  01 GRP-A.
      05 NUMBER1   PIC 9(3) VALUE 300.
  01 GRP-B.
      05 NUMBER1   PIC 9(3) VALUE 200.
  ADD CORRESPONDING GRP-A TO GRP-B.       # NUMBER1 OF B -> 500
  ```
- `GIVING`: Biến lưu kết quả.
- `ROUNDED`: Làm tròn kết quả nếu phần thập phân vượt quá định nghĩa của biến lưu trữ.
  - Cắt bỏ tất cả các chữ số thừa bên phải
  - Làm tròn số cuối cùng lên nếu chữ số tiếp theo >= 5
  - Nếu chữ số tiếp theo < 5 thì giữ nguyên.
- `ON SIZE ERROR`: Xử lý khi kết quả bị tràn dữ liệu.
- `NOT ON SIZE ERROR`: Xử lý khi kết quả không bị tràn dữ liệu.
- `END-ADD`: Từ khóa kết thúc lệnh ADD (thường dùng trong các cấu trúc lệnh lồng nhau).

  ```bash
  01 GP-1.
    02 A    PIC 9(2) VALUE 50.
    02 B    PIC 9(2) VALUE 60.
    02 C    PIC 9(2).
    02 D    PIC 9(2).
  ...
  ADD A TO B.                   # B = A + B = 10 (110 nhưng chỉ nhận 2 số sau)
  ADD A TO B ROUNDED.           # B = A + B = 60 (B lúc này là 10)
  ADD A TO B GIVING C D.        # C = d = A + B = 10 (110 nhưng chỉ nhận 2 số sau), A và B không đổi
  ADD A TO B ROUNDED ON SIZE ERROR DISPLAY "ON SIZE ERROR". # ON SIZE ERROR vì A + B - 110 vượt quá 9(2)
  ```

### SUBTRACT - Phép trừ

---

```cobol
SUBTRACT {CORRESPONDING/CORR} {Identifier1/literal1} FROM {identifier2} GIVING {Identifier3} {ROUNDED}
[ON SIZE ERROR imperative-statement-1]
[NOT ON SIZE ERROR imperative-statement-2]
END-SUBTRACT
```

Tương tự như ADD

```bash
01 GP-1.
    02 A    PIC 9(2) VALUE 50.
    02 B    PIC 9(2) VALUE +60.
    02 C    PIC 9(2).
    02 D    PIC 9(2).
...

SUBTRACT A FROM B.                      # B = B - A = 1{
SUBTRACT A FROM B ROUNDED.              # B = 4}  # Lúc này B = 1{
SUBTRACT A FROM B ROUNDED ON SIZE ERROR DISPLAY "ON SIZE ERROR". # DISPLAY NOTHING
SUBTRACT A FROM B GIVING C D.          # C = D = B - A = 40
```
